## üöÄ Getting Started

Follow these simple steps to run the gamezone locally on your machine.

### Prerequisites

You only need a modern web browser (like Chrome, Firefox, or Edge).

### Installation and Run

1.  **Clone the Repository (Optional):**
    ```bash
    git clone [https://github.com/Lavanya06-06/GAMEZONE.git](https://github.com/Lavanya06-06/GAMEZONE.git)
    ```
2.  **Download:** If you don't use Git, you can download the repository as a ZIP file.
3.  **Launch:** Open the **`index.html`** file in your preferred web browser. The hub will load immediately.
